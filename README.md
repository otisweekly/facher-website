# fächer Website

Marketing website for [fächer](https://facherapp.com) - the collection cataloging app.

## Pages

- `index.html` - Landing page
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service
- `support.html` - Support & FAQ

## Deployment

This is a static site that can be deployed to any hosting provider:

- **Vercel**: Connect repo and deploy automatically
- **Netlify**: Connect repo and deploy automatically
- **GitHub Pages**: Enable in repo settings
- **Namecheap**: Upload files via file manager or FTP

## Domain

Configure DNS at Namecheap to point `facherapp.com` to your hosting provider.
